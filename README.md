# Proximo Feriado

A simple Dockerfile to get the next holiday on Argentina 



### Pre Requisites

`docker 19.03` or later installed in your machine.

You can check the version with the following command.
``` 
$ docker --version
Docker version 19.03.2, build 6a30dfc
```
